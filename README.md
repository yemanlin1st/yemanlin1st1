== Description ==

WPSmart-Career-Profile is a WordPress plugin designed to enhance career and professional profiles by helping users create, structure, and improve profile content with AI-assisted suggestions.

The plugin is aligned with the broader PEFY-GG / PEFY-TECH public-safe governance logic for reusable, maintainable, risk-aware, and production-ready digital assets.

== Installation ==

1. Upload the `wpsmart-career-profile` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Add the shortcode `[wpscp_profile_form]` to the selected page or post.
4. Test the form with a non-production profile before public use.
5. Review privacy, consent, data storage, and user-facing notices before deployment.
